## Long-form description (project page, README hero, ~1200 chars)

> **YUCCA-FX 8-BIT** is a chiptune-flavored sound effect synthesizer built for game developers, chiptune musicians, and anyone who wants to crack out NES-style blips, lasers, explosions, and weird textural noise without installing anything.
>
> The synthesis vocabulary is the real one: Fourier-synthesized pulse waves with NES duty cycles (12.5 / 25 / 50 / 75 percent), 4-bit-stepped triangle via PeriodicWave, and real LFSR noise with both long and short metallic NES tap modes. Wrapped around the oscillator core is a draggable ADSR canvas, a filter with live frequency response visualization, an LFO with selectable waveform routing, a pitch envelope, drive saturation, delay with safety feedback fade, NES-quantized reverb, RetroVerb slapback, circuit-bent glitch effects with volume safety, and a mutant delay cascade that triggers full mutated copies of your patch at configurable intervals.
>
> The Lissajous Sequencer adds three independent euclidean rhythm tracks with deterministic per-step variation. Record N cycles to WAV or MP3 with automatic LCM-based loop length detection so the output loops cleanly.
>
> Single-patch export, 20-item export queue with batch rename, ZIP bundling, brick-wall limiter on by default, persistent state across sessions, URL sharing, BACK button with 32-deep history, tooltips on every control. Single HTML file, vanilla JS, no build, no dependencies, runs offline.
